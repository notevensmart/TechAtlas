# Roadmap

TechAtlas V1 is the foundation for a broader labour-market intelligence platform.

## Layer 1: Market Demand

V1 implements this layer:

- posting counts
- skill demand
- trend history
- city and role breakdowns
- salary coverage

## Layer 2: Skill Relationships

V1 implements ranked co-occurrence pairs. V2 can add:

- weighted skill graph table
- graph clustering
- skill-neighbour API
- Cytoscape.js graph explorer

## Layer 3: Career Pathways

Planned:

- skills-gap endpoint
- target-role recommendations
- role similarity clusters
- roadmap generator

Recommended progression:

1. Rule-based skill gaps.
2. Graph-based adjacent skills.
3. Listing/profile embedding similarity.

## Layer 4: Salary and Scarcity

V1 stores visible salary and guarded comparisons. Later work:

- scarcity score
- leverage score
- demand versus salary scatter plots
- rare/high-value skill detection

## Layer 5: Technology Evolution

V1 supports skill trend history. Later work:

- technology comparison mode
- timeline playback
- ecosystem shift analysis
- AI integration tracking across traditional software roles
